Install the DSD Constructor

Getting Ready

System requirements

  • PC running Microsoft Windows 7 or Later. The DSD Constructor runs in both 32- and 64-bit environments.

  • .NET 4.5.2 or later (included with recent Windows installations), and all updates and patches.

  • 120MB disk space to install the application.

Installing 


					
Click here to download and install the DSD Constructor

Updating


The DSD Constructor makes use of the ClickOnce Deployment method which enables it to self-update to the newest version. With this strategy, the DSD Constructor can be installed and run with minimal user interaction. The only requirement to trigger the update is to have internet access at the moment you launch the application.
