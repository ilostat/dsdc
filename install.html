Install the SDMX Constructor

Getting Ready

System requirements

  • PC running Microsoft Windows 7 or Later. The SDMX Constructor runs in both 32- and 64-bit environments.

  • .NET 4.7.2 or later (included with recent Windows installations), and all updates and patches.

  • 120MB disk space to install the application.

Portable Version


If you encounter any installation issues or require a standalone network version for your organization, you can obtain a downloadable zip file of this portable application. Please be aware that this portable version will not receive automatic updates, and we will only provide limited stable versions.

Updating


The SDMX Constructor makes use of the ClickOnce Deployment method which enables it to self-update to the newest version. With this strategy, the DSD Constructor can be installed and run with minimal user interaction. The only requirement to trigger the update is to have internet access at the moment you launch the application.
